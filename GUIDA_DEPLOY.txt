METODO FACILE (Interfaccia Web):
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
1. Nel repository appena creato, vedrai la scritta 
   "Quick setup â€” if you've done this kind of thing before"
   
2. Sotto, clicca sul link "uploading an existing file"

3. Trascina TUTTI questi file nella pagina:

   - .gitignore

4. In fondo alla pagina, scrivi nel campo "Commit changes":
